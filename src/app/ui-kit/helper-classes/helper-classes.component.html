کلاس های کمک کننده

فاصله

اختصاص دادن margin یا padding به یک عنصر یا یک زیر مجموعه از طرف آن با کلاس های کوتاه. شامل پشتیبانی از خواص فردی، تمام خواص، و ویژگی های عمودی و افقی است. تمام کلاس ها بر روی مقدار پیش فرض جهانی چند برابر هستند 1rem.

کلاس ها با استفاده از فرمت نامگذاری می شوند: {ویژگی}-{طرف}-{سایز}

  • m - برای کلاس های تعیین شدهmargin
  • p - برای کلاس های تعیین شده padding

این جدول حاوی کلاس برای اضافه کردن عناصر بالا، پایین، چپ، راست و حاشیه سمت همه است.

کلاس توضیح
.mt-{size} To set margin top use .mt-{size} class, where {size} is value between 0-5.
.mb-{size} To set margin bottom use .mb-{size} class, where {size} is value between 0-5.
.ml-{size} To set margin left use .ml-{size} class, where {size} is value between 0-5.
.mr-{size} To set margin right use .mr-{size} class, where {size} is value between 0-5.
.mx-{size} To set both margin left & right use .mx-{size} class, where {size} is value between 0-5.
.my-{size} To set both margin top & bottom use .my-{size} class, where {size} is value between 0-5.
.m-{size} To set margin 4 sides of the element use .ma-{size} class, where {size} is value between 0-5.

این جدول حاوی کلاس برای اضافه کردن عناصر به بالا، پایین، چپ، راست و تمام عرض جانبی است.

کلاس توضیح
.pt-{size} To set padding top use .pt-{size} class, where {size} is value between 0-5.
.pb-{size} To set padding bottom use .pb-{size} class, where {size} is value between 0-5.
.pl-{size} To set padding left use .pl-{size} class, where {size} is value between 0-5.
.pr-{size} To set padding right use .pr-{size} class, where {size} is value between 0-5.
.px-{size} To set both padding left & right use .px-{size} class, where {size} is value between 0-5.
.py-{size} To set both padding top & bottom use .py-{size} class, where {size} is value between 0-5.
.p-{size} To set padding 4 sides of the element use .pa-{size} class, where {size} is value between 0-5.

جایی که سایزیکی از:

  • 0 - for classes that eliminate the margin or padding by setting it to 0
  • 1 - (by default) for classes that set the margin or padding to $spacer-x or $spacer-y
  • 2 - (by default) for classes that set the margin or padding to $spacer-x * 1.5 or $spacer-y * 1.5
  • 3 - (by default) for classes that set the margin or padding to $spacer-x * 3 or $spacer-y * 3

حامیان محتوا

نمایش دادن

کلاس های کمک کننده محتوا نمایش، موقعیت، Z-شاخص، لبه ها، تراز عمودی، مکان نما، گزینه های سرریز و لیست را فراهم می کند.

یکی از این کلاس ها را اضافه کنید تا خواص نمایش های الکترونیک را تغییر دهید.

کلاس توضیح
.d-inline عنصر را به عنوان یک عنصر درونی رفتار می کند.
.d-inline-block عنصر را به عنوان عنصر inline-block رفتار می کند.
.d-block نیروی عنصر را به عنوان یک عنصر بلوک رفتار می کند.
.d-none نیروی عنصر را در همه نمایشگاه ها پنهان می کند.
موقعیت

یکی از این کلاس ها را برای تنظیم خواص موقعیت های Elemets اضافه کنید.

کلاس توضیح
.position-fixed تغییر موقعیت عنصر به ثابت.
.position-relative تغییر موقعیت عنصر به نسبی.
.position-absolute تغییر موقعیت عنصر به مطلق.
.position-static تغییر موقعیت عنصر به استاتیک.
.position-sticky تغییر موقعیت عنصر به چسبنده.
چیدمان عمودی

یکی از این کلاسها را برای تغییر خواص ترازبندی عمودی elemets اضافه کنید.

کلاس توضیح
.align-top تغییر مقطع عمودی عنصر به بالا.
.align-middle تغییر مقطع عمودی عنصر به وسط.
.align-bottom تغییر مقطع عمودی عنصر به پایین.
.align-baseline تغییر مقطع عمودی عنصر به خط مقدم.
.align-text-top بالای عنصر با بالای فونت عنصر والدین قرار دارد.
.align-text-bottom پایین عنصر با پایین فونت عنصر والدین هماهنگ است.
لبه ها

یکی از اینها را اضافه کنید تا لبه های عریض عناصر را حذف کنید، با عناصر مطلق، ثابت و نسبی موقعیت کار می کند

کلاس توضیح
.no-edge-top لبه بالایی عنصر را حذف می کند.
.no-edge-bottom لبه پایینی عنصر را حذف می کند.
.no-edge-left لبه چپی عنصر را حذف می کند.
.no-edge-right لبه راستی عنصر را حذف می کند.
سرریز

اضافه کردن یکی از این به کلاس برای تغییر سرریز از عنصر.

کلاس توضیح
.overflow-visible پیش فرض سرریز قطع نشده است. این در خارج از جعبه عنصر ارائه می شود.
.overflow-hidden سرریز قطع شده است و بقیه مطالب نامرئی هستند.
.overflow-auto اگر سرریز کپی شده باشد، برای مشاهده بقیه محتوا، یک اسکرول اضافه می شود.
.overflow-scroll سرریز کپی شده است، اما برای مشاهده بقیه محتوا، یک اسکرول اضافه شده است.
لیست

برای تغییر سبک لیست، یکی از اینها را به کلاس اضافه کنید.

کلاس توضیح
.list-unstyled هیچ نشانگر در مورد لیست نمایش داده نمی شود.
.bullets-inside مارکر و متن را کنار می گذارد. گلوله درون جریان محتوای نمایش داده می شود
.list-style-circle نشانگر مورد لیست را به دایره تنظیم کنید.
.list-style-square نشانگر مورد لیست را به مربع تنظیم کنید.

مرز ها

مرز اضافه کنید

یکی از اینها را به کلاس اضافه کنید تا مرز را در سمت مورد نیاز اضافه کنید.

کلاس توضیح
.border مرز یکپارچه را در هر طرف اضافه کنید.
.border-top اضافه کردن مرز بالایی 1px
.border-bottom پای مرزی 1px را اضافه کنید.
.border-left حاشیه جامد سمت چپ 1px را اضافه کنید
.border-right مرز راست را 1 px اضافه کنید.
حذف مرز

برای حذف مرز در سمت مورد نظر، یکی از اینها را به کلاس اضافه کنید.

کلاس توضیح
border-0 مرز را از هر طرف بردارید.
border-top-0 مرز را از قسمت بالا بردارید.
border-bottom-0 مرز را از قسمت پایین بردارید.
border-left-0 مرز را از قسمت چپ بردارید.
border-right-0 مرز را از قسمت راست بردارید.
عرض مرز

یکی از این ها را به کلاس اضافه کنید تا عرض عرض را در سمت مورد نظر تغییر دهید.

کلاس توضیح
.border-2 تعویض عرض را به 2 پیکسل در هر طرف.
.border-top-2 تعویض عرض را به 2 پیکسل در سمت بالا.
.border-bottom-2 تعویض عرض را به 2 پیکسل در سمت پایین
.border-left-2 تعویض عرض را به 2 پیکسل در سمت چپ
.border-right-2 تعویض عرض را به 2 پیکسل در سمت راست
.border-3 تعویض عرض را به 3 پیکسل در هر طرف
.border-top-3 تعویض عرض را به 3 پیکسل در سمت بالا
.border-bottom-3 تعویض عرض را به 3 پیکسل در سمت پایین
.border-left-3 تعویض عرض را به 3 پیکسل در سمت چپ
.border-right-3 تعویض عرض را به 3 پیکسل در سمت راست
حذف شعاع مرزی

یکی از اینها را به کلاس اضافه کنید تا شعاع مرزی را در قسمت مورد نظر حذف کنید.

کلاس توضیح
.no-border-top-radius حذف شعاع مرزی از طرف بالا.
.no-border-bottom-radius حذف شعاع مرزی از طرف پایین.
.no-border-top-left-radius حذف شعاع مرزی از طرف بالا و چپ.
.no-border-top-right-radius حذف شعاع مرزی از طرف بالا و راست.
.no-border-bottom-left-radius حذف شعاع مرزی از طرف پایین و چپ.
.no-border-bottom-right-radius حذف شعاع مرزی از طرف پایین و راست.

سایز

عرض و ارتفاع

برای تنظیم عرض محتوا، یکی از اینها را به کلاس اضافه کنید.

کلاس توضیح
.fit محتوای حداکثر عرض 100٪ را تنظیم کنید.
.half-width محتویات محتوا را 50٪ تنظیم کنید.
.full-width محتویات محتوا را 100٪ تنظیم کنید.
.full-height ارتفاع محتوا را 100٪ تنظیم کنید.
عرض ثابت

برای تنظیم پهنای ثابت محتوای، یکی از اینها را به کلاس اضافه کنید.

کلاس توضیح
.width-{size} مقدار محتوا ثابت در پیکسل اندازه انتخاب شده که در آن اندازه می تواند باشد 50, 100, 150, 200, 250, 300, 350, 400, 450, 500, 550, 600, 650, 700, 750 and 800. i.e .width-50
.width-{size}-per محتویات ثابت محدوده ثابت در درصد اندازه انتخاب شده که اندازه آن می تواند باشد 5%, 10%, 15%, 20%, 25%, 30%, 35%, 40%, 45%, 50%, 55%, 60%, 65%, 70%, 75% and 80%. i.e .width-5-per
ارتفاع ثابت

برای تنظیم ارتفاع ثابت محتوای، یکی از اینها را به کلاس اضافه کنید.

کلاس توضیح
.height-{size} تنظیم ارتفاع ثابت در پیکسل اندازه انتخاب شده که اندازه آن می تواند باشد 50, 100, 150, 200, 250, 300, 350, 400, 450, 500, 550, 600, 650, 700, 750 and 800. i.e .height-50
.height-{size}-per مقدار ثابت ارتفاع ثابت در درصد اندازه انتخاب شده که اندازه آن می تواند باشد 5%, 10%, 15%, 20%, 25%, 30%, 35%, 40%, 45%, 50%, 55%, 60%, 65%, 70%, 75% and 80%. i.e .height-5-per

عکس

برای تغییر اندازه تصویر و سایر ویژگی، یکی از اینها را به کلاس اضافه کنید.

کلاس توضیح
.bg-cover مقیاس تصویر پس زمینه را تا آنجا که ممکن است به طوری که منطقه پس زمینه به طور کامل توسط تصویر پس زمینه پوشیده شده است.
.bg-repeat تصویر پس زمینه را تکرار نکنید.
.bg-no-repeat تکرار تصویر پس زمینه
