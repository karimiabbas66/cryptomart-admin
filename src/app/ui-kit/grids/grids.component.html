بدون زاویه

زوایای بین ستونها در کلاسهای شبکه از پیش تعریف شده ما میتوانند با حذف شوند .no-gutters. این منفی را حذف می کند margins از جانب .row و افقی padding از تمام ستون های فرزند فوری

این کد منبع برای ایجاد این سبک هاست. توجه داشته باشید که لغو ستون به تنها ستون های فرزند اول محدود می شود و از طریق هدف قرار می گیرند ویژگی انتخاب .در حالی که این انتخاب کننده خاص تر را تولید می کند، می توان با استفاده از پلاگین های ستون هنوز هم سفارشی کرد آب و هوا.

نیاز به طراحی لبه به لبه؟ والدین .container یا .container-fluid.

  
      .no-gutters {
          margin-right: 0;
          margin-left: 0;
      
          > .col,
          > [class*="col-"] {
          padding-right: 0;
          padding-left: 0;
          }
      }

در عمل، در اینجا این است که چگونه به نظر می رسد. توجه داشته باشید می توانید همچنان از این استفاده کنید با تمام دیگر کلاسهای شبکه از پیش تعریف شده (از جمله عرض ستون، سطوح پاسخگو، ترتیب و موارد دیگر).

.col-12 .col-sm-6 .col-md-8
.col-6 .col-md-4
  
      <div class="row no-gutters">
          <div class="col-12 col-sm-6 col-md-8">.col-12 .col-sm-6 .col-md-8</div>
          <div class="col-6 col-md-4">.col-6 .col-md-4</div>
      </div>

.col-9
.col-4
